# Consignes projet

## Attribution / mentions « Claude » — INTERDIT

Ne JAMAIS faire apparaître « Claude », « Claude Code », « Anthropic » ni aucun
identifiant de modèle/session dans quoi que ce soit de versionné ou de public.
Cela concerne notamment :

- **Commits** : pas de trailer `Co-Authored-By: Claude ...`, pas de
  `Claude-Session:`, et l'auteur/committer doit être l'utilisateur (Rayan),
  jamais « Claude ».
- **Pull requests** : pas de footer « 🤖 Generated with Claude Code », pas de
  lien `claude.ai/code/session_...` dans le titre ni le corps.
- **Commentaires** (PR, issues, reviews) : pas de mention
  « _Generated by Claude Code_ » ni de lien de session.
- **Noms de branches** : pas de préfixe ni de segment `claude/...`.
- **Code, commentaires de code, docs** : aucune mention non plus.

En résumé : tout ce qui part vers GitHub doit être propre, au nom de
l'utilisateur, sans aucune trace d'outil d'assistance.
